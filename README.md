# ⭐ News Homepage

Proyecto de maquetación web basado en el desafío de Frontend Mentor **"News Homepage"**.  
El objetivo fue construir una página de inicio tipo portal de noticias completamente responsive, implementando layout moderno con CSS Grid, diseño adaptable y un menú móvil interactivo con JavaScript vanilla, aplicando manipulación del DOM, manejo de estado y accesibilidad web.

---

## 📌 Descripción

Este proyecto consiste en una página principal de noticias con diseño moderno, organizada en secciones principales como artículo destacado, barra lateral de noticias recientes y artículos en tendencia.

La interfaz incluye un menú de navegación responsive que se adapta a dispositivos móviles mediante un drawer lateral con overlay, el cual se abre y cierra dinámicamente mediante interacción del usuario.

Se puso especial énfasis en:

- Estructura semántica HTML  
- Diseño responsive con enfoque mobile-first  
- Layout avanzado con CSS Grid y Flexbox  
- Interactividad con JavaScript vanilla  
- Manejo de estado del menú lateral  
- Accesibilidad web mediante atributos ARIA  
- Experiencia de usuario (UX) mediante transiciones suaves  
- Código limpio, modular y mantenible

## 🎯 Características principales

- 📱 Diseño completamente responsive para mobile, tablet y desktop  
- 🍔 Menú hamburguesa con drawer lateral interactivo  
- 🌑 Overlay dinámico con control de interacción  
- 🔄 Manejo de estado del menú con JavaScript  
- 🎨 Animaciones suaves con `transition` de Tailwind CSS  
- 🧩 Imágenes optimizadas con `<picture>` (AVIF / WebP / JPG fallback)  
- ♿ Implementación de accesibilidad con `aria-expanded` y `aria-controls`  
- ⚡ Estructura moderna de proyecto con Vite  
- 🧱 Layout complejo usando CSS Grid

## 🧠 Aprendizaje

Durante este proyecto puse en práctica conceptos clave como:

- Manipulación del DOM con JavaScript  
- Uso de `addEventListener()` para eventos de interacción  
- Manejo de estado del menú con variables booleanas  
- Control dinámico de clases con `classList.toggle()`  
- Implementación de un drawer lateral (sidebar) responsive  
- Uso de overlay para mejorar UX en navegación móvil  
- Uso de atributos ARIA para accesibilidad (`aria-expanded`, `aria-controls`)  
- Estructuración de layouts complejos con CSS Grid  
- Optimización de imágenes con `<picture>` y múltiples formatos  
- Separación de lógica de UI y estructura HTML  
- Mejores prácticas en proyectos frontend modernos
